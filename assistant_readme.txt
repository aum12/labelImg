1. Instructions on how to use the tool. 

General steps from scratch
a.) Launch tool: 
    - Open terminal window
    - enter: cd ~/label_tool
    - enter: python labelImg.py
b.) Click 'Change default saved annotation folder' in Menu/File
c.) Click 'Open Dir'
d.) Click 'Create RectBox'
The annotation will be saved to the folder you specify

Hotkeys
Ctrl + r : Change the defult target dir which saving annotation files
Ctrl + n : Create a bounding box
Ctrl + s : Save
n : Next image
p : Previous image

**All details can be found https://github.com/tzutalin/labelImg. (The Hotkeys section should prove
useful).
